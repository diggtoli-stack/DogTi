# DogTI — 测试题目与结果逻辑

## 维度说明

每道题的选项各对应一个 MBTI 维度：

| 维度 | 含义 |
|------|------|
| E / I | 外向 / 内向 |
| S / N | 实感（具体）/ 直觉（想象） |
| T / F | 理性 / 感性 |
| J / P | 计划 / 随性 |

`calcResult()` 统计各维度得分，各取多数，组合出 MBTI 类型（如 INTJ）。

## 结果逻辑

```
e = E >= I ? "E" : "I"
s = S >= N ? "S" : "N"
t = T >= F ? "T" : "F"
j = J >= P ? "J" : "P"
type = e + s + t + j  // 例：INTJ
```

## 题目设计原则

- **不问具体行为**（刨坑、吃饭），而是通过狗狗视角让用户带入人的情感、情绪和态度
- **无「主人」相关题目**，避免用户出戏
- **前4题简单有趣**，让用户快速代入，不会纠结
- **最后一题固定**：理想的狗生（直接测 E/I/P）
